# FlowSuite
FlowSuite – SaaS starter (Next.js 16 + Tailwind) regroupant Composer Studio, Calendar Ops, Automation Hub et Clips Automation. Mode démo en mémoire, connecteurs OAuth YouTube/TikTok/Instagram/LinkedIn/X/Twitch, uploads chunkés, workflows BullMQ, dashboard complet prêt à déployer (Docker/Vercel/Render).
